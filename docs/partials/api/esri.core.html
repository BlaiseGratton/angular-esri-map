esri.core Module

The esri.core module includes services used by the directives in the esri.map module. These services can also be used directly by your own custom directives.

Loading Esri Modules

Use esriLoader to lazy load the Esri ArcGIS API or to load API modules.

Creating Map and Layer Instances

Use the esriMapUtils and esriLayerUtils services to help create instances of maps and layers.

Getting a Reference to the Map

Use the esriRegistry to store and retrieve map instances for use in parent controllers.
