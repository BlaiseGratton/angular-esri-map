esri.map Module

The esri.map module includes reusable directives for maps, layers, and related UI elements such as legend and info template.

Working with Maps

Use the esriMap directive allows you to place a map on the page. Use the esriLegend directive to place show a legend for the map.

Adding Layers to a Map

Use the esriFeatureLayer and esriDynamicMapServiceLayer directives to add layers to a map. Use the infoTemplate directive to format layer pop ups.
