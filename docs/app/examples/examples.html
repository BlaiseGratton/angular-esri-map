<esri-map/>

Examples showing how to use the angular-esri-map directives.

Simple map

Create a map by loading center, zoom, and basemap from an options object.

Webmap

Load a webmap from ArcGIS Online and show changes to center and zoom level via bound scope properties.

Set basemap

Shows how to use angular binding to link a select list to the basemap property of the map.

Set map center and zoom

Demonstrates two-way binding of the center, and zoom properties.

Feature layers

Loads two feature layers into the map and shows how to bind layer visibility to a check box.

Legend

Show a legend for the map.

Map events

Shows how to listen for events raised by the map directive and get a direct reference to the map object.

Registry Pattern

Shows how to get a reference to the map object by using the registry.

Additional Map Options

Shows how to load a map with additional map options.

Advanced Map Options

Shows how to load a map with more complex map options, including a custom Extent and Popup.
